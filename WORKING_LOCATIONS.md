## How to Use

1. **Open test page:** `c:\Users\HP\Desktop\Tossel-main\test_route.html`
2. **Enter Start:** Copy any name from above (e.g., "Hussein Dey")
3. **Enter Destination:** Copy another name (e.g., "Agha")
4. **Click Find Route**
5. **See detailed steps with transfers!**

## Tips

- Names are case-insensitive
- Partial matching works (e.g., "hussein" will match "Hussein Dey")
- Use exact names for best results
- Try nearby stations for faster routes
- System will show transfers automatically
